Lista Completa de Propriedades CSS

1. Propriedades de Texto

Propriedade Função Exemplo
color Define a cor do texto color: red;
Define a cor do texto como vermelha.
font-size Tamanho da fonte font-size: 16px;
Define o tamanho da fonte como 16px.
font-family Tipo de fonte (Arial, Verdana) font-family: Arial, sans-serif;
Define a fonte como Arial, com fallback para sans-serif.
font-style Faz o texto ser itálico font-style: italic;
Aplica itálico ao texto.
font-weight Define a espessura (negrito) font-weight: bold;
Aplica negrito ao texto.
letter-spacing Espaço entre letras letter-spacing: 2px;
Define o espaçamento entre as letras como 2px.
line-height Altura da linha do texto line-height: 1.5;
Define a altura da linha como 1.5 vezes o tamanho da fonte.
text-align Alinha o texto (esquerda, centro) text-align: center;
Centraliza o texto.
text-decoration Sublinhado, tachado, nenhum text-decoration: underline;
Aplica sublinhado ao texto.
text-transform Maiúsculas, minúsculas, capitalizado text-transform: uppercase;
Transforma o texto em maiúsculas.
white-space Controla espaços e quebras de linha white-space: nowrap;
Impede que o texto quebre a linha.
word-spacing Define o espaço entre palavras word-spacing: 5px;
Define o espaçamento entre as palavras como 5px.

2. Cores e Fundo

Propriedade Função Exemplo
background-color Cor de fundo background-color: #f0f0f0;
Define a cor de fundo como cinza claro.
background-image Imagem de fundo background-image: url('imagem.jpg');
Define uma imagem como fundo do elemento.
background-position Posição da imagem background-position: center;
Posiciona a imagem de fundo no centro.
background-size Tamanho da imagem background-size: cover;
Faz a imagem de fundo cobrir todo o elemento.
background-repeat Define se a imagem repete ou não background-repeat: no-repeat;
Não repete a imagem de fundo.
opacity Transparência (0 a 1) opacity: 0.5;
Define a opacidade do elemento como 50%.
box-shadow Sombra em torno do elemento box-shadow: 2px 2px 5px #888888;
Cria uma sombra com 2px de desvio, 5px de desfoque e cor cinza.

3. Bordas e Espaçamento

Propriedade Função Exemplo
border Borda completa (largura, tipo, cor) border: 1px solid #000;
Adiciona uma borda de 1px sólida preta.
border-radius Arredonda os cantos border-radius: 10px;
Arredonda os cantos com 10px de raio.
margin Espaço externo ao redor do elemento margin: 10px;
Adiciona 10px de espaço em todos os lados.
padding Espaço interno dentro do elemento padding: 10px;
Adiciona 10px de espaço dentro de todos os lados.

4. Posicionamento e Tamanhos

Propriedade Função Exemplo
width Largura do elemento width: 300px;
Define a largura do elemento como 300px.
height Altura do elemento height: 200px;
Define a altura do elemento como 200px.
position Define a posição do elemento position: relative;
Define a posição do elemento como relativa.

5. Flexbox (Layout Responsivo)

Propriedade Função Exemplo
display: flex Ativa o Flexbox display: flex;
Ativa o layout Flexbox para o elemento.
justify-content Alinhamento horizontal justify-content: center;
Alinha os itens no centro horizontalmente.
